Petunjuk Penggunaan

  • Klik angka dan operator untuk melakukan perhitungan
  • Gunakan tombol "DEL" untuk menghapus angka terakhir
  • Gunakan tombol "C" untuk menghapus semua perhitungan
  • Klik "=" untuk melihat hasil perhitungan
  • Klik ikon matahari untuk mengubah tema
Dibuat dengan HTML, CSS, dan JavaScript
